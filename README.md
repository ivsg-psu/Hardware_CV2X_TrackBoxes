## About The Hardware

<!--[![Product Name Screen Shot][product-screenshot]](https://example.com)-->

This repo is purely about the hardware set and specifications. At each installation pole, a sealed, weather-proof cabinet is used to house and protect all communication and computing hardware. This cabinet integrates the on-board computer, wireless communication modules, uninterruptible power supply (UPS), and C-V2X devices into a single, secure platform designed to ensure reliable operation under varying environmental and field conditions.

For software installation of CV2X, please visit <a href="https://github.com/ivsg-psu/Hardware_CV2X_Commsignia">Hardware CV2X Commsignia</a>.

For test and data analysis of CV2X, please visit <a href="https://github.com/ivsg-psu/FieldDataCollection_VisualizingFieldData_PlotCV2X">FieldDataCollection_VisualizingFieldData_PlotCV2X</a>
